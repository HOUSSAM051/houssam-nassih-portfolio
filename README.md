# Portfolio Houssam Nassih

Un portfolio moderne et professionnel présentant le parcours et les projets de Houssam Nassih, étudiant en ingénierie des réseaux et IT.

## 🚀 Fonctionnalités

- **Design moderne et responsive** : Interface élégante qui s'adapte à tous les écrans
- **Navigation fluide** : Menu de navigation fixe avec défilement smooth
- **Animations interactives** : Effets de scroll, transitions et animations subtiles
- **Formulaire de contact** : Interface functional pour envoyer des messages
- **Timeline éducation** : Présentation visuelle du parcours académique
- **Galerie de projets** : Mise en valeur des projets académiques avec tags technologiques
- **Compétences détaillées** : Barres de progression animées pour les langues et technologies

## 📁 Structure du projet

```
portfolio/
├── index.html          # Page principale du portfolio
├── styles.css          # Styles CSS avec design moderne
├── script.js           # Fonctionnalités JavaScript interactives
```

## 🔧 Personnalisation

### Modifier les couleurs
Éditez les variables CSS dans `styles.css` :

```css
:root {
    --primary-color: #2563eb;    /* Couleur principale */
    --secondary-color: #1e40af;  /* Couleur secondaire */
    --accent-color: #3b82f6;     /* Couleur d'accent */
}
```

### Ajouter un projet
Modifiez la section projets dans `index.html` :

```html
<div class="project-card">
    <div class="project-icon">
        <i class="fas fa-votre-icone"></i>
    </div>
    <h3>Titre du projet</h3>
    <p class="project-date">Année</p>
    <p>Description du projet...</p>
    <div class="project-tags">
        <span>Technologie 1</span>
        <span>Technologie 2</span>
    </div>
</div>
```

### Modifier les informations personnelles
Mettez à jour les informations dans les sections correspondantes du fichier `index.html`.
